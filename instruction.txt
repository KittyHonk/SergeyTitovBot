При необходимости изменения интервала обнуления столбца AlreadyCheck (Работает только для MySql) использовать скрипт: 

alter EVENT new_day on SCHEDULE every 1 DAY; 

Где "1 DAY" можно изменить на необходимый интервал, вместо DAY принимаются MONTH, WEEK, HOUR, MINUTE. 
Пример обновления раз в 2 недели: 

alter EVENT new_day on SCHEDULE every 2 WEEK; 
_____________________________________________

Т.к. столбец AlreadyCheck отвечает за возможность выхода на работу и в случае проблем с ним приводит к отказу прикладываю скрипт экстренного сброса:

update worker set AlreadyCheck = 0
where AlreadyCheck = 1;

Так же добавлена команда для бота, которая выполняет этот скрипт удаленно, через самого бота: /resetalreadycheck
